# WindowsでVPNに接続するとネットに出られなくなった

メインのWindows機を新調てからはじめてVPNに接続したら, インターネットに出られなくなった. 
ゲートウェイがおかしいのかと思っていろいろ試してみたが, 結局VPNの設定を変更したら治ったので, 手順を記録しておく. 

## 解決手順

1. Windowsの設定からネットワークとインターネット → VPNの設定にアクセス
2. 使用しているVPNの設定にアクセスし, **その他のVPNプロパティ**欄にある編集をクリック
3. ネットワークタブから, インターネットプロトコル バージョン4 (TCP/IPv4) のプロパティにアクセス
4. プロパティのウィンドウから詳細設定にアクセス
5. 「リモートネットワークでデフォルトゲートウェイを使う」のチェックボックスを外す
6. VPNをつなぎなおす
